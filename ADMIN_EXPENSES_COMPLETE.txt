WORKFLOWS:

BULK APPROVE:
1. Select expenses using checkboxes
2. Click "Approve Selected"
3. All selected approved
4. List refreshes

BULK REJECT:
1. Select expenses using checkboxes
2. Click "Reject Selected"
3. Dialog opens for reason
4. Enter reason
5. Click "Reject"
6. All selected rejected with reason
7. List refreshes

INDIVIDUAL ACTIONS:
1. Click Edit to modify
2. Click Approve to approve single
3. Click Reject to reject with reason
4. Click Delete to remove
5. Click Download to get receipt

SELECT ALL:
1. Click "Select All" checkbox
2. All expenses selected
3. Perform bulk action
4. Click "Select All" again to deselect
