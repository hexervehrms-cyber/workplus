WHAT WAS FIXED
================================================================================

Issue: Check In, Break, and Meeting buttons were not working

Root Cause: 
- Frontend was sending wrong user ID format (user.id instead of user.userId)
- Frontend was not sending orgId parameter
- User object didn't have userId and orgId properties

Solution:
- Updated User interface to include userId and orgId
- Updated login function to extract userId from JWT token
- Updated login function to extract orgId from JWT token
- Updated all attendance handlers to use correct parameters
