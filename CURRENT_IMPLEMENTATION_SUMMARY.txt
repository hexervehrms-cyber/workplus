function, which prevented admin employee CRUD operations from working.

- Function broadcasts employee updates to all connected clients
- Supports 'created', 'updated', 'deleted' actions

File Modified: backend/server.js (lines 237-248)

================================================================================
EMITTER MIDDLEWARE FUNCTIONS
================================================================================

All the following functions are now properly defined in backend/server.js:

1. req.emitAttendanceUpdate(attendance, orgId)
   - Broadcasts attendance changes to organization
   - Used by: attendance routes

2. req.emitLeaveUpdate(action, leaveRequest, orgId)
   - Broadcasts leave request changes
   - Used by: leave routes

3. req.emitDashboardUpdate(action, type, data, orgId)
   - Broadcasts dashboard updates
   - Used by: all CRUD routes

4. req.emitActivityUpdate(activity, orgId)
   - Broadcasts activity log entries
   - Used by: all routes that modify data

5. req.emitNotification(notification, userId, orgId)
   - Sends notifications to users or organization
   - Used by: all routes that need to notify users

6. req.emitEmployeeUpdate(action, employee, orgId) [NEWLY ADDED]
   - Broadcasts employee management updates
   - Used by: employee routes (create, update, delete)
