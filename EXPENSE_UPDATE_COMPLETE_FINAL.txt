================================================================================
                            FORM STRUCTURE
================================================================================

REQUIRED FIELDS (*)
  1. Claim Title - Custom title for the expense
  2. Category - Type of expense (Travel, Food, Office, etc.)
  3. Amount (₹) - Expense amount in rupees
  4. Date - Date of the expense

OPTIONAL FIELDS
  1. Description - Additional details about the expense
  2. Upload Receipt - Receipt file (PDF, PNG, JPG)

BUTTONS
  - Cancel: Closes dialog without saving
  - Submit Claim (Create): Submits new expense
  - Update Claim (Edit): Updates existing expense

================================================================================
                            USER WORKFLOWS
================================================================================

WORKFLOW 1: CREATE WITH TITLE AND RECEIPT
  1. Click "Add Expense" button
  2. Enter Claim Title: "Client Meeting Lunch"
  3. Select Category: "Food"
  4. Enter Amount: "500"
  5. Select Date: "02/05/2026"
  6. (Optional) Add Description
  7. Upload Receipt
  8. Click "Submit Claim"
  Result: Expense created with title in list

WORKFLOW 2: CREATE WITH TITLE ONLY (NO RECEIPT)
  1. Click "Add Expense" button
  2. Enter Claim Title: "Office Supplies"
  3. Select Category: "Office"
  4. Enter Amount: "1000"
  5. Select Date: "02/05/2026"
  6. Skip Receipt Upload (it's optional)
  7. Click "Submit Claim"
  Result: Expense created without receipt

WORKFLOW 3: UPDATE EXPENSE TITLE
  1. Find pending expense in list
  2. Click "Edit" button
  3. Dialog opens with current data
  4. Modify Claim Title: "Updated Title"
  5. (Optional) Upload receipt if needed
  6. Click "Update Claim"
  Result: Title updated in list

WORKFLOW 4: ADD RECEIPT LATER
  1. Find pending expense without receipt
  2. Click "Edit" button
  3. Upload Receipt
  4. Click "Update Claim"
  Result: Receipt added to existing expense
