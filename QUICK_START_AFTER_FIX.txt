IF YOU ENCOUNTER ERRORS:
================================================================================

Error: "req.emitEmployeeUpdate is not a function"
- Solution: Make sure you restarted the backend server after the fix

Error: "Cannot connect to MongoDB"
- Solution: Check your MongoDB connection string in backend/.env
- Verify MongoDB is running

Error: "Socket.IO connection failed"
- Solution: Check that the backend server is running
- Check browser console for connection errors

Error: "Unauthorized" when creating/editing employees
- Solution: Make sure you're logged in as an admin
- Check that your JWT token is valid
