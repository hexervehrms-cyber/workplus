================================================================================
TROUBLESHOOTING
================================================================================

PROBLEM: "Name, email, and password are required"
SOLUTION: Fill in all three fields

PROBLEM: "Password must be at least 6 characters"
SOLUTION: Enter password with 6 or more characters

PROBLEM: "User with this email already exists"
SOLUTION: Use a different email address

PROBLEM: "Invalid email format"
SOLUTION: Use format like: user@domain.com

PROBLEM: User appears in list but can't login
SOLUTION: Verify email and password are correct

WHAT HAPPENS AFTER CREATION
================================================================================

1. User is stored in MongoDB database
2. Password is hashed using bcrypt
3. User appears in the users list
4. User can login with email and password
5. User gets access based on their role:
   - Employee: Employee dashboard
   - Manager: Manager dashboard
   - HR: HR dashboard
   - Admin: Admin dashboard
   - Accountant: Accounting dashboard
   - Super Admin: Super admin dashboard
