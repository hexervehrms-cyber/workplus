SECURITY FLOW
================================================================================

User Password
    │
    ├─→ Frontend: Masked input (••••••••)
    │
    ├─→ Sent to backend via HTTPS
    │
    ├─→ Backend: Hash with bcrypt (12 rounds)
    │   Original: "SecurePass123"
    │   Hashed:   "$2b$12$..."
    │
    ├─→ Store hashed password in database
    │
    ├─→ Never store plain password
    │
    ├─→ Never log password
    │
    ├─→ On login: Compare input with hash
    │   Input: "SecurePass123"
    │   Hash:  "$2b$12$..."
    │   Match: ✓ Login successful
    │
    └─→ User authenticated
